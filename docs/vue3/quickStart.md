## 3.安装依赖

### 1.安装node环境
可以去[node官网下载](https://nodejs.org/zh-cn/download/)，也可以使用[nvm node管理工具](https://github.com/nvm-sh/nvm#installing-and-updating)，大家自行选择

### 2.安装yarn环境
后面版本升级会采用pnpm，可以提前做好安装准备
```shell
npm install yarn -g
npm install pnpm -g
```
### 3.安装依赖
```shell
yarn install
```
![](/images/vue3/quickStart/quickStart3.png)

## 4.启动项目
